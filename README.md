## Лабораторная №1. Линейная регрессия

- **Соревнование:** [Ferma Challenge на Kaggle](https://www.kaggle.com/t/a8bb3f56df904646ab67263af86e562e)
- **Цель:** построить линейную модель, прогнозирующую годовой удой коровы (`milk_yield_kg`) и минимизирующую MAE.
- **Что делаем:** анализируем датасет фермы, готовим признаки, обучаем baseline `LinearRegression`, затем экспериментируем с регуляризацией, добавляем фичи и сравниваем метрики.
- **Артефакты:** шаблон ноутбука и структура лежат в `lab01-linear-regression/`.
